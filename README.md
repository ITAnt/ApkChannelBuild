# ApkChannelBuild
多渠道打包，纯java生成一个渠道名的空文件，以便在Android App中用代码动态获取渠道名

umeng-analytics-v6.0.7.jar 、utdid4all-1.0.4.jar 这两个jar文件是UMeng统计的，只是为了测试代码添加渠道的方式，在本项目中并没有什么用
